## What is MCP?

MCP (Model Context Protocol) is an open protocol that allows Claude to interact with external tools and data sources. MCP servers can provide:

- Access to external APIs and services
- Database connections
- File system operations
- Custom business logic

## Creating an MCP Server

See the [MCP documentation](https://modelcontextprotocol.io/) for details on building MCP servers.

## Submitting Your MCP Server

1. Create a directory for your server: `plugins/mcp-servers/your-server-name/`
2. Include a `plugin.json` with metadata
3. Include a `README.md` with installation and usage instructions
4. Submit a pull request

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full guidelines.
